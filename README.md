# Primer Ejemplo con Selenium IDE 
[![GitHub issues](https://img.shields.io/github/issues/ualhmis/seleniumIDEejemploClase.svg)](https://github.com/ualhmis/seleniumIDEejemploClase/issues)
1. Instalar [Katalon Recorder](https://www.katalon.com/resources-center/blog/katalon-automation-recorder/) como plugin de Firefox o Chorme

2. Abrir Katalon Recorder en Firefox o Chrome. 
	
3. Cargar la Test Suite: _HmisEnWebUAL.html_
  
5. Lanzar los tests

## Contribuye

1. Haz un fork del repositorio a tu propia cuenta en GitHub (https://github.com/ualhmis/seleniumIDEejemploClase/fork)
2. Clona el repositorio de tu cuenta a tu computadora local (`git clone https://github.com/xxxxxxx/seleniumIDEejemploClase`)
3. Crea tu rama nueva de característica con el nombre tema en cuestion (`git checkout -b mi-nueva-rama`)
4. Guarda los cambios con commit (`git commit -am 'Añadida la característica'`).
5. Haz un push de tu rama hacia tu fork (`git push origin mi-nueva-rama`)
6. Crea un nuevo Pull Request en tu fork.
